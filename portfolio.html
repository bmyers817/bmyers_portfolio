Scratch Project

Using scratch, I created a game. My partner, Alex Hughes, and I created this game in a couple of days! Using our minds, we made a historical satire on Cold-War Era communism. In the game, you play as Joseph Stalin, and you travel to three distinct locations in the Eastern World to spread communism and defeat capitalism. In this program, we implemented numerous amounts of algorthims. In fact, the movement for the main character, "Stalin", utilizes several algorithms. Additionally, the character's attack actions utilize an algrithm to determine the direction of the attack. In addition to algorithms, this project made use of a multitude of variables so as to set the health for both the character and all of the bosses. Additionally, some of the enviornments are interactive, such as the China level, where the boss attempts to throw a block labeled "capitalism" at you. This block can be hit out of the boss's hands and reset back to its original position. Whilst this project only took a few days, I am thoroughly satisfied with the end product. I believe it meets all of the acceptable standards for the project and provides an interesting game for all to enjoy.

Another language that I have learned is MIT APP Inventor. APP Inventor is a free-to-use program that utilizes blocks of code that can mixed around similar to scratch to design programs and apps for android systems. If you want to code in APP Inventor, click on the logo to visit the site!


MIT APP Inventor Project

Using APP Inventor, I created a game. My partner, Shaun Boscovich, and I created a game with APP Inventor in a small time frame. We made an 8-bit arcade style western game for two players. In the game, player one and player two move up and down the screen at different rates. One player shoots faster while the other moves up and down faster. Each player can be shot 4 times before death. Once one player is "dead", the game ends and the other player wins. In this program, we implemented a variety of algrothims for the movement of the characters as well as for their shooting. Additionally, we have a music screen implemented so that the user can customize the ambiance for the game. In addition to the movement algrorithms, this program utilizes variables for the health of the players. While the game is rather simple in execution, I believe that it serves its purpose to entertain extensively.

Text Adventure Project

Using Repl.it, I created a text-based adventure. Using repl.it, my partner, Deonarain Mamo, and I created a text-based adventure about Vladimir Lenin. This adventure, titled "Lenin's Rise to Power", follows Vladimir Lenin's formation of the communist party in Russia. In the adventure, the user is faced with some of the key choices that Lenin faced during his lifetime. Depending on the decisions that the user makes, a different outcome can occur. Each outcome branches down to different endings, creating a flow for the adventure. Multiple algorithms were added to the program to assist with the decisions of the user.
